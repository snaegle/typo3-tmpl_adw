Benutzerverwaltung
==================
An der Webseite der Akademie der Wissenschaften zu Göttingen arbeiten mehrere Personen mit. Die Benutzerverwaltung beinhaltet ein komplexes Rechtemanagement um festzulegen welche Möglichkeiten jede Redakteurin bzw. jeder Redakteur haben soll.
Bitte beachten Sie: Dieses Kapitel kann nur einen Ausschnitt auf das umfangreiche Rechtemanagement in TYPO3 bieten. Bitte bedenken Sie, dass Veränderungen bei den Nutzergruppen, Nutzern, dem Dateimanagement u. ä. Auswirkungen auf alle Redakteure haben.
Testen Sie immer zunächst auf dem Development-Server https://dev.adw-goe.de/typo3/, wenn Sie neue Nutzergruppen und Nutzer anlegen möchten! Prüfen Sie die Einstellungen und vollziehen Sie dann erst auf dem Live-Server die Einstellungen.
